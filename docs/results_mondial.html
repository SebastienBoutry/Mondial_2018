Les buts en détail

Au total, nous avons 145 buts (dont 20 sur pénalty et 10 buts contre son camp (CSC)).
Le treemap permet de visualiser la distribution des buts selon le poste, la partie du corps et le nom du joueur.

Information sur le poste du buteur :

  • ATT : attaquant
  • MIL : milieu
  • DEF : défenseur
  • GAR : gardien

Information sur le code couleur :

  • vert : pied droit
  • bleu : pied gauche
  • rouge : tête
